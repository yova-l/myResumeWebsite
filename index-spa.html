fileMover: herramienta para fotógrafos/videógrafos

Desarrollo propio

Propósito: Desarrollado para mejorar la productividad en la edición de fotos al automatizar el proceso de recuperar las copias de mayor calidad (RAWs) a partir de una selección de fotos en baja calidad, reduciendo una tarea manual de varias horas a un solo clic.

Tecnologías utilizadas: Python, Tkinter, POO, desarrollo de GUI para Windows y Linux, desarrollo de sitios web a partir de una plantilla, implementación en GitHub Pages, manipulación de archivos y pruebas unitarias.

Desarrollé este proyecto para optimizar mi flujo de trabajo en la edición de fotos. Cuando viajo y tomo fotografías, a menudo enfrento la tediosa tarea de seleccionar fotos y luego localizar manualmente copias de mayor calidad, lo cual puede llevar horas. Mi programa automatiza este proceso con un solo clic, ahorrando considerablemente tiempo y esfuerzo.
